Teams Project


Year: 2023

University of Bristol

3 Months







Flow chart for the grass instancing architecture



  • Creating a system for instancing foliage, using indirect instancing and compute shaders to keep all data and instancing logic on the GPU.
  • Game was run on Valve Index, so all shaders had to be optimised to work on VR.
  • Instanced grass positions sampled from a grid in front of the camera. Various optimisations implemented like variable density, level of detail and front-to-back render ordering to reduce overdraw.
  • Grass and other instancing systems culled on the GPU via compute shader using frustrum and occlusion culling (4:31).
  • Translucent shader created for foliage with support for wind and texture atlases, including injected support for instancing (3:18).
  • Fog (3:35) and cloud shader from previous projects.
  • Procedural asset scattering on the GPU, including a distance field generator to scatter depending on the distance to obstacles (4:50).
  • Achieved 80% grade, making it the top team in the cohort.

Screnspace Fluids


Year: 2022



  • Screenspace fluids by rendering to a separate buffer and blurring depth and normals.
  • Used in a variety of VFX effects, examples show lava and fire using the effect on Unity's particle systems.

Raymarched Clouds


Year: 2021



  • Volumetric clouds rendered in a raymarched post-process effect. Optimised for realtime, lighting effects and dynamic weather.
  • Includes a 'godray' effect, using both the depth mask and a cloud mask to render godrays.
