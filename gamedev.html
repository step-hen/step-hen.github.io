Teams Project


Year: 2023

University of Bristol

5 Months



    Created in Unity Engine in a team of 6, developed for Valve Index.
  • Extended xNode UI to create a procedural generation pipeline and implemented many of the nodes. Ran the pipeline by traversing the graph.
  • Procedurally generated terrain using Unity's Terrain System with OpenTopography. Created buildings by running Blender headless, sending building footprints from OpenStreetMap, running Buildify and exporting the result.
  • Creating a system for instancing foliage and grass, frustrum/occlusion culling and instancing all on the GPU.
  • Accelerated nodes in the pipeline with the use of compute shaders.
  • Achieved 80% grade, making it the top team in the cohort.
